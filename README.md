# BookFiler Module - MySQL
Module containg MySQL connectivity, SQLite3, and HTTP.

## Implementation

| Technology | Library |
|:-- |:-- |
| MySQL Client | libmariadbclient |
| SQLite3 | sqlite3 |
| HTTP Client | curl |
